# NFYLL Visual Identity & Design Proposal

**Date:** 2026-05-25
**Source:** design-pass agent run during nfyll.org polish session

## Approved palette

| Role | Hex | Name |
|---|---|---|
| Primary | `#1B4D7E` | River Blue |
| Primary deep | `#103657` | Deep River |
| Primary tint / page bg | `#E8F0F7` | Sky Wash |
| Accent | `#F2823C` | Sunshine Orange |
| Secondary accent | `#F4C24B` | Sand Gold |
| Ink | `#1A2733` | Slate Ink |
| Muted | `#5A6B7B` | Tide Gray |
| Hairline | `#D6DEE6` | Hairline |

## Typography

- **Display**: Fraunces (Google Fonts, variable opsz + wght)
- **Body**: Inter (Google Fonts, variable)

## Hero (light mode, light Sky Wash background)

- Hero illustration: stylized NE-Florida map (St. Johns River + coastline) with 8 member-club towns marked as Sunshine Orange dots
- Eyebrow above H1: `OFF-SEASON` or `SPRING 2026 · WEEK N OF 8`
- H1 (Fraunces 700, opsz 144): rotates by state
- Primary CTA: Sunshine Orange filled button
- Secondary CTA: text link with arrow

## Schedule card

Light card on Sky Wash page, 1px Hairline border, 8px radius.
- Sunshine-Orange date chip top-left
- Matchup line: Inter 600
- Venue: "Field N · Julington Creek" (lean into field number, not venue name — all games same venue)
- Division pill at bottom: tinted to division (4v4=Gold, 7v7=Orange, 10v10=Blue)
- Hover: border becomes Sunshine Orange, slight translateY + soft shadow

## Schedule page additions

1. Sticky day-jumper pill row at top (Feb 22 · Mar 1 · ... )
2. Club filter dropdown
3. Quieter section breaks between days

## Anti-patterns to AVOID

- Dark mode / near-black surfaces (that's FILC's territory)
- Navy + green (FILC palette)
- Stock action photos
- Countdown timers
- Standings/leaderboards/MVP (rec league, no public scores)
- Win/loss badges on cards
- 5-6 col auto-fit grids that look like icon walls
- Bootstrap-style feature rows w/ FontAwesome
